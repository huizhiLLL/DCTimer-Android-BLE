# 项目简介

DCTimer Android 是基于 DCTimer-Android 的二次开发版本，当前主要围绕蓝牙硬件计时能力完善展开。

目前项目重点覆盖两条硬件主线：

- 智能魔方
- 蓝牙计时器

# 当前支持范围

当前版本已接入部分智能魔方与蓝牙计时器设备，具体请以支持设备列表为准。

# 系统要求

- 支持系统：Android
- 最低系统版本：Android 8.0
- 最低 API Level：26
- 推荐使用较新的 Android 设备以获得更稳定的蓝牙连接体验
