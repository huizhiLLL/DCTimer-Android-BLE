# DCTimer-BLE 软件关键信息

日期：`2026-05-15`

## 基本信息

- 软件名称：`DCTimer-BLE`
- 当前版本：`2.2`
- Android 包名：`com.dctimer.ble`
- APK 输出名：`DCTimer-BLE-v2.2.apk`

## 底层系统支持

- 支持平台：`Android`
- 当前仅维护 Android 版本，暂不维护 iOS、桌面端或 Web 端。
- 构建基线：`AndroidX + AGP 8.9.2 + Gradle 8.11.1 + JDK 17`
- `compileSdk / targetSdk`：`35`
- `minSdk`：`14`

## 蓝牙设备支持

### 智能魔方

- `MoYu32`
- `QiYi / Tornado V4 智能版`
- `GAN v2 / v3 / v4`

### 蓝牙计时器

- `QiYi Smart Timer`

## 当前支持边界

- 当前硬件主线只分为 `智能魔方` 和 `蓝牙计时器` 两类。
