# 当前已接入的智能魔方

- `MoYu32` 系列（魔域智能）
- `QiYi`/`Tornado V4 智能版` （奇艺智能和风ai）
- `GAN` 智能魔方 `v2 / v3 / v4`

## 智能魔方蓝牙计时当前怎么工作

- 连接成功后，应用会同步当前魔方状态和电量，并持续接收转动事件。
- 当前计时不是“连接即起表”，而是按当前打乱目标自动判断何时进入 `READY`。
- 页面进入 `READY` 后，第一次真实复原转动才真正开始计时。
- 魔方恢复 solved 状态后会自动停表、自动保存成绩、自动进入下一轮打乱，不再弹出无惩罚 / `+2` / `DNF` 的复核框。

## 当前已经补上的智能魔方交互

- 计时器页面右下角会实时显示智能魔方 3D 状态
- 主页面和状态弹窗都提供了“重置魔方”入口，便于快速回到还原态。
- 打乱公式区域已经支持按正确打乱进度做渐隐/变色提示。
- 如果用户在按公式打乱时走偏，页面会异步计算并显示“最短纠错路径 + 剩余打乱”；后续沿纠错路径前进时会增量推进，而不是每一步都整段重算。

## 为什么扫不到任何蓝牙设备？

确保蓝牙和定位都已经打开，并尝试重试扫描

## 为什么列表没有显示我的智能魔方？

确保智能魔方处于唤醒状态并为连接其他客户端
