# 当前已接入的蓝牙计时器

- `QiYi Smart Timer`

## 智能魔方蓝牙计时当前怎么工作

- 连接成功后，应用会同步当前魔方状态和电量，并持续接收转动事件。
- 当前计时不是“连接即起表”，而是按当前打乱目标自动判断何时进入 `READY`。
- 页面进入 `READY` 后，第一次真实复原转动才真正开始计时。
- 魔方恢复 solved 状态后会自动停表、自动保存成绩、自动进入下一轮打乱，不再弹出无惩罚 / `+2` / `DNF` 的复核框。

## 当前已经补上的智能魔方交互

- 计时器页面右下角会实时显示智能魔方 3D 状态
- 主页面和状态弹窗都提供了“重置魔方”入口，便于快速回到还原态。
- 打乱公式区域已经支持按正确打乱进度做渐隐/变色提示。
- 如果用户在按公式打乱时走偏，页面会按实际错误步序逆序取逆，显示“纠错提示 + 剩余打乱”；同面连续两步会自动合并成 `X2`，连续两个 `X2` 会自动抵消。若纠错步数超过 `10` 步，则不再继续给出纠错序列，改为提示先复原魔方，并限制提示频率。

## 当前蓝牙硬件入口怎么区分

- `智能魔方`：只扫描并连接当前已适配的智能魔方设备
- `蓝牙计时器`：只扫描并连接当前已适配的蓝牙计时器设备

## 蓝牙计时器当前怎么工作

- 当前首个正式接入的是 `QiYi Smart Timer`
- 蓝牙计时器链路已经和智能魔方链路拆开，不再复用魔方状态图与魔方业务逻辑
- `QiYi Smart Timer` 当前按 `READY -> RUNNING -> STOPPED` 状态驱动主页面计时与成绩保存

## 为什么扫不到任何蓝牙设备？

确保蓝牙和定位都已经打开，并尝试重试扫描

## 为什么列表没有显示我的智能魔方？

确保智能魔方处于唤醒状态并为连接其他客户端
