## 本地使用方式

### 方案 1：项目根目录放置 `key.properties`

复制 `key.properties.example` 为 `key.properties`，填写以下字段：

- `storeFile`
- `storePassword`
- `keyAlias`
- `keyPassword`

然后执行：

```powershell
.\rel
```

### 方案 2：命令行临时传入

```powershell
.\gradlew.bat assembleRelease `
  -PreleaseStoreFile=你的签名文件路径 `
  -PreleaseStorePassword=你的签名文件密码 `
  -PreleaseKeyAlias=你的别名 `
  -PreleaseKeyPassword=你的密钥密码
```
